# Welcome to the Innioasis Y1 Themes community site GitHub Repository

Adding Themes:

1. Fork the InnioasisY1Themes Repository
2. Create a pull request
3. Upload your theme's folder to the main directory of the repository - /yourthemename
4. Add a line to themes.json detailing the folder, name of the theme and your author name + URL (or that of the author who's theme you're uploading)
5. Submit your pull request
This will then be merged and your theme added to the site when a repository admin merges the pull into the main repository.

[There are a number of repository admins/maintainers](https://github.com/y1-community/InnioasisY1Themes/people) that can assist with uploads and merge pull requests, this is of course done voluntarily by community members as a hobby so please be patient, once "merged" new theme submissions will usually appear on the website within 5 minutes.

# minimal

A minimalist theme and color-matched wallpapers designed specifically for the Innioasis Y1.

## Installation

### Innioasis Y1 Theme

1. Connect your Innioasis Y1 to your computer using a USB cable.
2. Locate the `Themes` folder at the root of your device's internal storage.
3. Create a new folder named `minimal` (or another name of your choice) inside the `Themes` folder.
4. Transfer all files from this repository into that folder (e.g., `/Themes/minimal/`).
5. Use the device's built-in theme settings to apply the theme.

### Setting Wallpaper Colors

To ensure wallpaper colors are correctly recognized by the system apps:

1. Locate the `y1-colors-wallpapers` folder in this repository.
2. Move the `y1-colors-wallpapers` folder directly to the **root** of your device's internal storage.
3. On your Innioasis Y1, open the **Photos** app or navigate to the **Wallpapers** configuration in the system settings.
4. Select your preferred color from the moved folder to set it as your desktop and global wallpaper.

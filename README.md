### Adding themes to the gallery

Simply visit https://themes.innioasis.app/upload.html to add new your themes to the Gallery, if your theme comes in multiple variations, upload the zip of the theme's main variation then upload zips with the other variations, for example if your theme uses different mask images for the menus, or backgrounds for each variation include the entire theme contents inside each zip, with their variation images already in place and simply specify what the variants are called for listing on the gallery with a drop-down for variation selection.

Upload Theme ZIP

Use this page to submit a .zip that contains one or more themes. The server creates a GitHub PR, then existing validation + auto-merge workflows decide whether it can merge.

What happens next

  • Your ZIP is uploaded through a Cloudflare Pages Function (/api/upload-theme).
  • The function creates a bot branch + PR in y1-community/InnioasisY1Themes.
  • PR validation checks for dangerous files and theme structure.
  • If valid, auto-merge can complete and zip processing on main extracts theme folders.

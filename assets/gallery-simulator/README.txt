Optional vendored React simulator (Theme Maker / ThemeDisplay)

The live Theme Maker app bundles the full device UI in its Vite build. This folder is reserved for a future checked-in UMD bundle (React + ThemeDisplay) if you build the editor locally and copy dist assets here.

The gallery site ships with y1-theme-preview.js + gallery-loaded-theme.js for static previews and LoadedTheme-shaped data without a separate npm build step.
